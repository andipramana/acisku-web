FAQ

Pertanyaan yang sering ditanyakan.

Apakah Acisku gratis?
Ya, sepenuhnya gratis. Tidak ada biaya langganan.
Apakah data keuangan saya aman?
Setiap baris data dilindungi Row Level Security, jadi hanya akun kamu yang bisa membacanya. Kamu juga bisa memilih menyimpan data di server Supabase milikmu sendiri, bukan cuma server bawaan Acisku.
Kenapa belum ada di Google Play atau App Store?
Acisku sedang dalam proses pengajuan ke Google Play. Untuk saat ini, install langsung lewat file APK di halaman ini — begitu sudah tersedia di Play Store, tautannya akan diperbarui di sini.
Bisa dipakai di iPhone?
Untuk saat ini Acisku baru tersedia untuk Android. Dukungan iOS masih dalam pertimbangan.
